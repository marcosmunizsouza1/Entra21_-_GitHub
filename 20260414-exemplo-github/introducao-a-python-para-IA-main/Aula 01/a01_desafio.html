Caixa Registradora — programa de vendas simples
Variáveis, operações, input(), print() e f-strings. Python puro, sem bibliotecas.
Contexto do desafio Você foi contratado por uma pequena lanchonete para criar um programa de caixa registradora básica. O programa deve perguntar o nome do cliente, a quantidade e o preço de até 3 itens, calcular o subtotal de cada item, o total da compra, aplicar um desconto de 10% se o total passar de R$ 50,00, e exibir um cupom fiscal formatado no terminal.
Requisitos — marque ao concluir
Critérios de avaliação
Variáveis com nomes descritivos em português
input() com conversão correta (int ou float)
Cálculos usando variáveis, não números fixos
f-strings com formatação :.2f para valores monetários
Saída organizada e legível (cupom fiscal)
Bônus: separador visual com print("="*30)
Bônus: exibir se houve desconto ou não

Entrega e dica de estrutura
Arquivo aula01_desafio.ipynb no Colab com todas as células executando sem erro. Mostre ao docente digitando um pedido completo ao vivo.

Sugestão de estrutura:
1. Pedir nome do cliente
2. Pedir quantidade e preço do item 1 → calcular subtotal1
3. Repetir para item 2 e item 3
4. Calcular total = subtotal1 + subtotal2 + subtotal3
5. Verificar se total > 50 (por enquanto, calcule desconto sempre — o if será na Aula 02)
6. Exibir o cupom com todos os dados formatados
