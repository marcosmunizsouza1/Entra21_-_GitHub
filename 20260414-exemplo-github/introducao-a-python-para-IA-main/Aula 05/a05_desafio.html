Sistema de Gestão de Estoque — modular e reutilizável
7 funções com responsabilidades claras. Nenhuma função faz mais de uma coisa.
Contexto do desafio Uma loja de eletrônicos precisa de um sistema de gestão de estoque em Python. Você deve construir o programa inteiro usando funções — cada operação é uma função separada. O programa deve: adicionar produtos ao estoque, verificar disponibilidade, aplicar descontos, gerar relatório de estoque baixo e calcular o valor total do inventário.
Requisitos — marque ao concluir
Critérios de avaliação
Cada função tem UMA responsabilidade clara
Docstring em todas as funções
Parâmetros com valor padrão onde faz sentido
Funções retornam valores (nao apenas imprimem)
Função principal chama as outras de forma organizada
Bônus: função que retorna múltiplos valores
Bônus: tratar divisao por zero na função de desconto

Estrutura base — use como ponto de partida
estoque = [] # lista global de produtos

Funções a implementar:
1. adicionar_produto(nome, preco, quantidade) → adiciona dict ao estoque
2. buscar_produto(nome) → retorna o dict ou None
3. verificar_disponibilidade(nome, qtd_desejada=1) → retorna True/False
4. aplicar_desconto(produto, percentual=10) → retorna preco com desconto
5. estoque_baixo(minimo=5) → retorna lista de produtos com qtd abaixo do minimo
6. valor_total_inventario() → retorna soma de preco*qtd de todos os produtos
7. relatorio_completo() → chama as outras e exibe tudo formatado
