Gerenciador de Biblioteca — catálogo com listas e dicionários
Lista de dicionários, busca, filtro e relatório formatado.
Contexto do desafio Uma biblioteca municipal precisa de um programa para gerenciar seu acervo. O sistema deve armazenar livros como lista de dicionários, permitir buscar um livro pelo título, listar todos os livros disponíveis (nao emprestados), calcular o total de páginas do acervo e exibir um relatório organizado por categoria.
Requisitos — marque ao concluir
Critérios de avaliação
Lista com pelo menos 5 dicionários de livros
Busca por título usando for + if + break
Filtro de disponíveis com acumulador em lista
.items() usado para iterar dicionários
Relatório formatado com f-string e separadores
Bônus: agrupar por categoria com dicionário de listas
Bônus: função para adicionar novo livro ao catálogo

Estrutura dos dados — use como base
Cada livro deve ser um dicionário com estas chaves:

{"titulo": "...", "autor": "...", "categoria": "...", "paginas": 0, "disponivel": True}

Crie pelo menos 5 livros variando categorias (ex: Ficção, Técnico, História) e disponibilidade.

Funcionalidades a implementar:
1. Buscar livro pelo título (for + break)
2. Listar todos os disponíveis (disponivel == True)
3. Somar total de páginas do acervo
4. Contar livros por categoria (dicionário acumulador)
