Relatório de Vendas — análise com laços
for, while, acumuladores e break. Dados reais, saída formatada.
Contexto do desafio Você é analista júnior em uma loja. O gestor quer um relatório de vendas do mês: total vendido, média diária, melhor dia de venda, quantos dias bateram a meta de R$1.000 e o primeiro dia que ultrapassou R$1.500 (para identificar o pico inicial). Tudo a partir de uma lista de vendas diárias.
Requisitos — marque ao concluir
Critérios de avaliação
for percorrendo a lista de vendas
Acumulador para total e contador para dias acima da meta
Variável para guardar o maior valor (melhor dia)
for + break para encontrar o primeiro pico
Relatório formatado com :.2f e :.1f
Bônus: usar enumerate() para mostrar o número do dia
Bônus: while para pedir ao usuário uma nova meta
