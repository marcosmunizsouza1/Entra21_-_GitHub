Análise de Dados de Monitoramento Climático
Arrays 2D, vetorização, estatísticas por eixo e detecção de anomalias.
Contexto do desafio Uma empresa de energia solar precisa analisar dados de irradiação solar de 6 estações de monitoramento ao longo de 30 dias. Você recebeu os dados como array NumPy 2D e deve: calcular estatísticas por estação, detectar dias de anomalia (irradiação muito abaixo da média), normalizar os dados para comparação entre estações e gerar um relatório resumido.
Requisitos — marque ao concluir
Critérios de avaliação
Array 2D com shape (30, 6) criado corretamente
Estatísticas calculadas com axis=0 (por estacao)
Deteccao de anomalias com indexacao booleana
Normalizacao vetorizada sem loop
argmax/argmin usados para identificar melhores/piores
Bonus: Z-score para comparar anomalias entre estacoes
Bonus: funcao Python que recebe o array e retorna o relatorio

Dados do desafio — copie para o Colab
import numpy as np
np.random.seed(42)
dados = np.random.uniform(100, 900, (30, 6))
dados = np.round(dados, 1) # W/m² de irradiacao
# Shape: (30 dias, 6 estacoes)

Tarefas:
1. Calcule média, desvio padrão, min e max de cada estação (axis=0)
2. Identifique a melhor e a pior estação em média de irradiação
3. Detecte dias com irradiação abaixo de 200 W/m² (possível cobertura de nuvens)
4. Normalize cada estação para [0, 1] usando vetorização
5. Calcule quantos dias cada estação operou acima de 700 W/m²
